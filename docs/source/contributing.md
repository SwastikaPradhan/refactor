# Contributing

- First fork the [repository](https://github.com/isidentical/refactor)
- Then clone your fork (`git clone git@github.com:<your username>/refactor`)
- Checkout to the `refactor` directory (`cd refactor`)
- Install pre-commit hooks, `pre-commit install`
- Add the `upstream` as a remote (`git remote add upstream https://github.com/isidentical/refactor.git`)
- Create a branch from `upstream/master` (`git fetch upstream; git checkout -b <your branch name> upstream/master`)
- Make the changes you need and push them to your branch
- Create a PR (don't forget to add an adequate description of what you tried to achieve)
